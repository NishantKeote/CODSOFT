# AeroPods - Modern Product Landing Page

A sleek, highly responsive, and visually striking product landing page built from scratch to showcase next-generation audio technology. This project was designed to demonstrate modern CSS layout techniques, crisp typography, intuitive navigation, and clean interactive elements.

🚀 **[View Live Demo](https://6a16a0bec999767d404e38ba--gilded-choux-9fb36d.netlify.app/)**

---

## 📱 Features

- **Sticky Navigation:** A modern, blurred backdrop-filter navigation bar that stays accessible as you scroll.
- **Fluid Grid Layouts:** Leverages CSS Grid for structural dual-column layouts (Hero and Specs sections) and flexible three-column feature cards.
- **Custom Interactive Elements:** Features smooth hover micro-interactions, an animated morphing product background vector, and a glowing pulsing UI element.
- **Form UI Concept:** Includes an elegant, minimalistic newsletter subscription interface with active focus states.
- **Fully Responsive Design:** Tailored with fluid media queries to snap seamlessly into an intuitive single-column layout on mobile and tablet screens.

---

## 🛠️ Tech Stack Used

- **HTML5:** Semantic architecture (`<nav>`, `<header>`, `<section>`, `<footer>`) for optimized SEO and accessibility.
- **CSS3:** Custom native CSS variables (tokens for colors, spacing, and animations), Flexbox, CSS Grid, and custom `@keyframes` animations.
- **Font Awesome:** Scalable vector icons for clean visual cues across the feature set and contact section.
